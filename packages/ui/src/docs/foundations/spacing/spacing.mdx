## 使用方法

### CSS変数

```css
.card {
  padding: var(--spacing-4);
  margin-bottom: var(--spacing-6);
}

.section {
  padding-block: var(--spacing-16);
}
```

### Tailwind CSS

<p>Tailwindのデフォルトスペーシングユーティリティがそのまま使用可能です。</p>

```tsx
<div className="mb-6 p-4">
  <section className="py-16">コンテンツ</section>
</div>
```
