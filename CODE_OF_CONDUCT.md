# 行動規範

## 目的

このプロジェクトは、すべての参加者にとって安全で歓迎される環境を提供することを目指しています。

## 期待される行動

- 敬意を持ったコミュニケーション
- 建設的なフィードバック
- 異なる意見や経験への理解
- コミュニティの成長への貢献

## 許容されない行動

- ハラスメント、差別的な発言
- 個人攻撃、侮辱
- 他者のプライバシーの侵害
- その他、不適切と判断される行為

## 違反の報告

行動規範に違反する行為を目撃した場合は、Issue で報告してください。すべての報告は真摯に対応します。

## 適用範囲

この行動規範は、Issue、Pull Request、コメント、その他プロジェクトに関連するすべての場に適用されます。
